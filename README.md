<h2>âš™ï¸ Requirements & Setup</h2>

<p>
  Most graphics programs use the <strong>BGI graphics library</strong>. You will need a
  compatible environment such as:
</p>

<ul>
  <li>Turbo C/C++ (with graphics support)</li>
  <li>WinBGIm library (for modern compilers)</li>
  <li>Code::Blocks with graphics setup</li>
</ul>

<p>
  Make sure to configure your IDE or compiler to include the graphics library before
  compiling and running the programs.
